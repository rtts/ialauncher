Download IA Launcher

Step 1: Download DOSBox

IA Launcher uses DOSBox, so install this DOS emulator first. Click here to go to the DOSBox download page and click the big green download button.

Step 2: Download IA Launcher

Now that you have installed DOSBox, there are multiple ways to install IA Launcher.

1. Windows

Click here to download the 64-bit Windows installer and run it. This will install IA Launcher for your current user (i.e., no administrator privileges required). Take note of the installation directory, however, because currently the installer does not create a Start menu item. You will have to locate ialauncher.exe manually after the installation is complete.

2. macOS

Open up a terminal and type the following command: 

$ sudo pip3 install ialauncher
This will install IA Launcher directly from PyPi.

3. Linux

If you are a GNU/Linux user, you probably just want access to the source code and figure it out yourself :-)

Click here to go to the GitHub page.
