Download IA Launcher

Step 1: Download DOSBox

IA Launcher uses DOSBox, so install this DOS emulator first. Click here to go to the DOSBox download page and click the big green download button.

Step 2: Download IA Launcher

Now that you have installed DOSBox, there are multiple ways to install IA Launcher.

Windows

Click here to download the 64-bit Windows installer. After downloading, doubleclick the file to run the installer. It will install IA Launcher for all users and add an entry to the Start menu.

MacOS/Linux

Open up a terminal and type the following command: 

$ pip3 install ialauncher
This will install IA Launcher directly from PyPi.
